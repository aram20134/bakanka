# Баканское озеро
Разработан с использованием MUI. 
Реализовано:
- Бронирование беседок с оплатой (эквайринг)
- Написание отзывов пользователями
- Наличие админ панели (статус заказов, бронирование, фильтрация отзывов, управление галереей)
